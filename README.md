# 🏥 Healthcare Data Analysis (SQL Project)

## 📌 Objective

Analyze healthcare data to uncover insights related to hospital performance, patient behavior, and revenue generation.

---

## 🛠 Tools Used

* SQL (MySQL)
* Joins, Group By, Window Functions

---

## 📂 Dataset Overview

The dataset contains information about:

* Patients
* Doctors
* Treatments
* Hospital revenue

---

## 🔍 Key Business Questions

* Which treatments generate the highest revenue?
* Who are the top-performing doctors?
* Which patients contribute the most to revenue?
* How does hospital performance vary?

---

## 📊 Key Insights

* Identified top revenue-generating treatments
* Found high-value patients contributing major revenue
* Compared doctor performance based on treatments and revenue
* Highlighted trends in hospital operations

---

## 📈 SQL Techniques Used

* JOINs for combining multiple tables
* GROUP BY for aggregation
* Window Functions for ranking and analysis
